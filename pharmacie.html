Configuration Firebase

Connecter la base de données partagée

Suivez ces étapes pour partager les données entre vos 2 postes (gratuit)

1
Allez sur console.firebase.google.com → Créez un projet pharmagest → Menu gauche : Realtime Database → "Créer une base" → Mode test → Activer
2
⚙️ Paramètres du projet → "Général" → "Vos applications" → icône </> Web → Nommez l'app → Copiez apiKey, projectId et databaseURL
3
Collez les 3 valeurs ci-dessous → Connecter. Même opération sur le 2ème poste avec les mêmes valeurs.

❌ Connexion échouée. Vérifiez vos informations.
